RELIFE - An Interactive Montage

五月 28, 2018

RELIFE - 1 / 40320


Editing is a magical thing.
Simply by changing the order of clips, we can change the narrative logic and the emotional traits of the film.

Based a fraction of thought, I worked with 林欣畅(Asteros) of China Central Academy of Fine Arts, created this interactive montage. In order to give more power of control for the audiences and give more possible outcomes for the story. This montage contains 8 swappable clips, viewer can change the order of the clips via replacing the cubes with markers.
In theory, there are A(8,8) = 1×2×3×4×5×6×7×8 = 40320 different combinations, which means 40320 different films.


My contribution to the work is that I wrote the interactive software and handled the physical structure of the installation.

I used OpenCV to identify the cubes with markers, and Unity3D to handle the sorting logic and the video stream.


You can check my github repo for the release version.
